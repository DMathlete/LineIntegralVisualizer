MA204 — Line Integral Visualizers

Interactive 3D tools for Block IV: Vector Calculus

Scalar Line Integral
$\displaystyle \int_C f\, ds$
Visualize the integral of a scalar function $f(x,y,z)$ along a curve $C$ as the area of a curtain. Includes Riemann sum animation, arc-length mode, and 2D / 3D curve presets.
Vector Line Integral
$\displaystyle \int_C \mathbf{F}\cdot d\mathbf{r}$
Visualize the work done by a vector field $\mathbf{F} = \langle P, Q, R\rangle$ along an oriented curve. Toggle field arrows, tangent projections, a signed work curtain, and a full field grid.
